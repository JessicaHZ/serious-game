# 🪴 Vivero Virtual: Serious Game de Botánica

Un Serious game interactivo de escritorio desarrollado para enseñar los fundamentos del cuidado de plantas reales mediante la simulación. 

Este proyecto utiliza la metodología **LM-GM (Learning Mechanics - Game Mechanics)** para traducir conceptos botánicos (como el riego y la fertilización) en mecánicas de juego intuitivas y recompensantes.

## ✨ Características Principales

* Mecánicas de riego, abono y poda con consecuencias directas en la salud de la planta.
* Indicadores de estado basados en cambios de *sprites* (Pixel Art) y barras de progreso, evitando la sobrecarga de información numérica.
* Sistema de experiencia, niveles de jugador y rachas diarias para fomentar la constancia.
* Estilo visual isométrico en Pixel Art, paleta de colores cálida y diseño horizontal (16:9) optimizado para PC.
* Construido bajo el patrón de diseño MVC (Modelo-Vista-Controlador) para garantizar una alta cohesión y un bajo acoplamiento entre la interfaz y la lógica de negocio.

## 🛠️ Tecnologías Utilizadas

* **Core:** Electron (Node.js + Chromium)
* **Frontend:** HTML5 Semántico, CSS3 (Flexbox/Grid)
* **Lógica:** JavaScript (Vanilla ES6+ con Módulos)
* **Persistencia:** LocalStorage / JSON

## 🚀 Requisitos Previos

Para ejecutar este proyecto en tu entorno local, necesitas tener instalado:
* [Node.js](https://nodejs.org/) (incluye npm)
